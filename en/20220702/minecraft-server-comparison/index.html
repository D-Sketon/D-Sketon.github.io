и®ЎиҪҜжқӮи°Ҳ

дј—жүҖе‘ЁзҹҘпјҢBugjumpеңЁMCй«ҳзүҲжң¬дёӢзҡ„дјҳеҢ–зЁҖзғӮпјҢеҰӮжӯӨиҫЈйёЎзҡ„дјҳеҢ–еҜ№дәҺжғіиҰҒејҖжңҚдҪҶжүӢеӨҙдёҠеҚҙеҸӘжңү1c2gжҲ–жҳҜ2c4gжёЈжңәзҡ„жңҚдё»жқҘиҜҙжҳҜдёҖдёӘйқһеёёеӨ§зҡ„иҙҹжӢ…гҖӮ
еҜ№дәҺspigotпјҢpaperзӯүйқһеҺҹзүҲз«Ҝзҡ„дјҳеҢ–MCBBSдёҠе·Із»ҸжңүйқһеёёеӨҡзҡ„её–еӯҗпјҢдјҳеҢ–зҡ„ж–№ејҸеҹәжң¬дёҠеҸҜд»ҘеҲҶдёәдёүзұ»пјҡ

  • дҝ®ж”№bukkit.ymlгҖҒspigot.ymlе’Ңpaper.ymlзӯүй…ҚзҪ®
  • еўһеҠ еҢәеқ—еҚёиҪҪе’Ңжё…зҗҶжҺүиҗҪзү©зӯүдјҳеҢ–жҸ’д»¶
  • дҝ®ж”№JavaеҗҜеҠЁеҸӮж•°е’ҢJavaзүҲжң¬

дҪҶе®һйҷ…дёҠпјҢдҝ®ж”№й…ҚзҪ®ж–Үд»¶дә§з”ҹзҡ„ж•Ҳжһңе№¶дёҚжҳҺжҳҫпјҢдё”иҝҮеҲҶзҡ„дҝ®ж”№еҸӮж•°дјҡеҜ№жёёжҲҸдҪ“йӘҢдә§з”ҹеҪұе“ҚгҖӮиҖҢжңүе…іеҢәеқ—еҚёиҪҪгҖҒжё…зҗҶжҺүиҗҪзү©гҖҒз”ҹзү©йҷҗеҲ¶зӯүдјҳеҢ–жҸ’д»¶еңЁй«ҳзүҲжң¬дёӢеҸҜд»ҘиҜҙжҳҜеҮ д№ҺжІЎжңүгҖӮ
JavaеҗҜеҠЁеҸӮж•°е…¶е®һеҜ№дәҺжңҚеҠЎеҷЁжқҘиҜҙйқһеёёйҮҚиҰҒпјҢдҪҶзҪ‘з»ңдёҠеҜ№дәҺеҗҜеҠЁеҸӮж•°зҪ‘дёҠзҡ„иҜҙжі•дј—иҜҙзә·зәӯгҖӮжң¬дәәдјҡеңЁжңӘжқҘеҜ№дёҚеҗҢзҡ„еҸӮж•°иҝӣиЎҢжөӢиҜ•пјҢж•¬иҜ·жңҹеҫ…гҖӮ
жң¬ж–ҮйҖҡиҝҮдёүдёӘжөӢиҜ•жқҘзІ—з•Ҙең°жҺўз©¶дёҖдёӢJava8е’ҢJava11дёӢHotspotе’ҢOpenj9иҷҡжӢҹжңәеҜ№дәҺMinecraftй«ҳзүҲжң¬жңҚеҠЎз«Ҝspigotз«Ҝзҡ„жҖ§иғҪеҪұе“Қ
жң¬ж¬ЎжөӢиҜ•йҮҮз”Ёзҡ„MinecraftзүҲжң¬дёә1.16.5пјҢжүҖд»Ҙе№¶дёҚж”ҜжҢҒJava17
SpigotзүҲжң¬дёә3096a-Spigot-9fb885e-af1a232
JavaеҗҜеҠЁеҸӮж•°йҮҮз”ЁжңҖз®ҖеҚ•зҡ„и®ҫзҪ®ж–№ејҸпјҢд»…д»…и®ҫзҪ®еҲқе§Ӣе Ҷе’ҢжңҖеӨ§е Ҷзҡ„еӨ§е°Ҹ

JavaйҖүз”Ёд»ҘдёӢеӣӣдёӘзүҲжң¬пјҡ

  • Openj9 jre8
  • Hotspot jre8
  • Openj9 jre11
  • Hotspot jre11

ең°еӣҫз§Қеӯҗпјҡ1145141919810
ејҖжңҚзі»з»ҹпјҡUbuntu20.04
жңҚеҠЎеҷЁй…ҚзҪ®пјҡи…ҫи®Ҝдә‘2c4g6m
жөӢиҜ•еҲҶдёәдёүдёӘйғЁеҲҶпјҡж— жҸ’д»¶ејҖжңҚжөӢиҜ•пјҢи·‘еӣҫжөӢиҜ•е’ҢеӨҡжҸ’д»¶ејҖжңҚжөӢиҜ•

ж— жҸ’д»¶ејҖжңҚжөӢиҜ•

иҜҙжҳҜж— жҸ’д»¶е®һйҷ…дёҠиҝҳжҳҜеӯҳеңЁдёҖдёӘжҸ’д»¶пјҡEssentialsX-2.19.0пјҢиҜҘжҸ’д»¶д»…д»…з”ЁдәҺиҝҗиЎҢ/gcе‘Ҫд»ӨиҺ·еҫ—tpsе’ҢAllocated memory
жөӢиҜ•ж–№жі•пјҡеңЁеӯҳжЎЈе·Із»Ҹз”ҹжҲҗеҗҺдҪҝз”ЁеӣӣдёӘдёҚеҗҢзҡ„JavaзүҲжң¬еҗҜеҠЁmcжңҚеҠЎеҷЁпјҢдҪҝз”Ёи…ҫи®Ҝдә‘зӣ‘жҺ§йқўжқҝиҺ·еҫ—CPUе’ҢеҶ…еӯҳеҚ з”Ёжғ…еҶөпјҢеҗҢж—¶дҪҝз”Ё/gcе‘Ҫд»ӨиҺ·еҫ—tpsе’ҢжёёжҲҸеҶ…Allocated memory

еңЁжөӢиҜ•зҡ„иҝҮзЁӢдёӯдёҚиҝҗиЎҢе…¶д»–зЁӢеәҸпјҢжӯӨж—¶и…ҫи®Ҝдә‘зӣ‘жҺ§йқўжқҝзҡ„ж•°жҚ®еҸҜд»Ҙиҝ‘дјјдёәmcжңҚеҠЎеҷЁзӢ¬еҚ 

CPUеҚ з”ЁзҺҮ

йқһеёёжҳҺжҳҫпјҢHotspotзӣёжҜ”дәҺOpenj9жқҘиҜҙCPUеҚ з”ЁзҺҮжӣҙдҪҺпјҢиҝҷд№ҹеҸҚеә”еңЁдәҶејҖжңҚж—¶й—ҙдёҠпјҡHotspotејҖжңҚеқҮеңЁ30-40sе·ҰеҸіпјҢиҖҢOpenj9ејҖжңҚж—¶й—ҙеқҮеңЁ60sд»ҘдёҠгҖӮ
жЁӘеҗ‘еҜ№жҜ”Java8е’ҢJava11пјҢеңЁиҷҡжӢҹжңәзӣёеҗҢзҡ„жғ…еҶөдёӢдёӨиҖ…е·®и·қе№¶дёҚжҳҺжҳҫпјҢдҪҶиҝҳжҳҜеҸҜд»ҘзңӢеҮә11жҜ”8еҚ з”ЁзҺҮиҰҒй«ҳдёҖдәӣгҖӮ

еҶ…еӯҳеҚ з”Ё

йқһеёёжҳҺжҳҫпјҢOpenj9еңЁеҶ…еӯҳдёҠе®ҢзҲҶHotspotпјҢиҝҷд№ҹз¬ҰеҗҲOpenj9зҡ„е®ҡдҪҚпјҡд»ҘзүәзүІдёҖйғЁеҲҶжҖ§иғҪзҡ„еүҚжҸҗдёӢжһҒеӨ§йҷҚдҪҺеҶ…еӯҳеҚ з”ЁзҺҮгҖӮ
жӯӨеӨ–зӣёеҗҢиҷҡжӢҹжңәдёӢ11еҶ…еӯҳеҚ з”ЁзҺҮиҰҒжҜ”8иҰҒй«ҳеҮәи®ёеӨҡгҖӮ

TPS

еҜ№дәҺTPSпјҢеӣӣдёӘдёҚеҗҢзүҲжң¬еқҮеңЁејҖжңҚеҲқжңҹеҸ‘з”ҹдәҶжҳҺжҳҫзҡ„дёӢйҷҚпјҢдҪҶйғҪиғҪд»ҘеҮ д№ҺзӣёеҗҢзҡ„йҖҹеәҰжҒўеӨҚгҖӮ

Allocated memory

иҜҘж•°жҚ®жқҘжәҗдәҺ/gcжҢҮд»Өд№ӢдёӯпјҢдёӘдәәж„ҹи§үиҜҘеҸӮж•°иҝҳжҳҜжҜ”иҫғиҝ·зҡ„пјҢд»…дҫӣеҸӮиҖғгҖӮ

и·‘еӣҫжөӢиҜ•

жөӢиҜ•ж–№жі•пјҡеңЁејҖжңҚ5еҲҶй’ҹеҗҺйҰ–е…Ҳиҝһз»ӯи·‘еӣҫ2еҲҶй’ҹпјҲзәҰ2600ж јпјҢеҲӣйҖ йЈһиЎҢпјүпјҢжҺҘзқҖжҢӮжңә13еҲҶй’ҹ
жҜҸж¬ЎйЈһиЎҢж–№еҗ‘еқҮзӣёеҗҢпјҢжөӢиҜ•е®ҢжҲҗеҗҺдјҡе°ҶеӯҳжЎЈйҮҚзҪ®дҝқиҜҒжҜҸж¬ЎжөӢиҜ•йғҪдјҡйҮҚж–°з”ҹжҲҗең°еҪў

еҶ…еӯҳеҚ з”Ё

Openj9 jre11еңЁиҝҷйҮҢеҸ‘з”ҹдәҶж„ҸеӨ–пјҢеңЁи·‘еӣҫ1еҲҶй’ҹж—¶еҶ…еӯҳжһҒйҖҹйЈҷеҚҮпјҢCPUд№ҹеҲ°дәҶ100%пјҢжңҚеҠЎеҷЁеҗҺеҸ°жҸҗзӨәoverloadпјҢдёӘдәәи®ӨдёәиҝҷеҸӘжҳҜдёӘеҲ«зҺ°иұЎпјҢдҪҶиҝҷд№ҹиҜҙжҳҺдәҶOpenj9еҸҜиғҪеҮәзҺ°зҡ„дёҚзЁіе®ҡжғ…еҶөгҖӮ
и·‘еӣҫж—¶зҡ„еҶ…еӯҳе’ҢйқҷжҖҒжөӢиҜ•е”ҜдёҖдёҚеҗҢзҡ„жҳҜHotspot jre8пјҢеңЁи·‘еӣҫе®ҢжҲҗеҗҺеҶ…еӯҳд»Қз„¶дёҠеҚҮпјҢдҪҶеҲ°дәҶ2000MеҗҺи§ҰеҸ‘дәҶеҶ…еӯҳеӣһж”¶пјҢеҶ…еӯҳдёӢйҷҚиҮі1680Mд»ҘдёӢпјҢиҝҷз”ҡиҮіжҜ”ејҖжңҚеҲқжңҹзҡ„еҶ…еӯҳиҝҳиҰҒдҪҺгҖӮз»ҸиҝҮеҸҚеӨҚжөӢиҜ•еҗҺзЎ®и®ӨдәҶиҜҘзҺ°иұЎе№¶дёҚжҳҜеҒ¶з„¶гҖӮ

TPS

еӣӣдёӘзүҲжң¬TPSеқҮдјҡдёӢйҷҚпјҢдҪҶз”ұдәҺHotspotеҗҜеҠЁйҖҹеәҰзәҰжҳҜOpenj9дёӨеҖҚпјҢжүҖд»ҘTPSдёӢйҷҚж—¶й—ҙзӮ№д№ҹз•Ҙеҫ®жҸҗеүҚгҖӮ

жҖ»з»“

Openj9зӣёжҜ”дәҺHotspotжҖ§иғҪдјҡжңүз•Ҙеҫ®зҡ„жҚҹеӨұпјҢе…·дҪ“иЎЁзҺ°дёәејҖжңҚйҖҹеәҰж…ўзәҰдёӨеҖҚпјҢCPUеҚ з”ЁзҺҮдјҡжӣҙй«ҳгҖӮдҪҶз”ұжӯӨеёҰжқҘзҡ„жҳҜеҶ…еӯҳзҡ„еӨ§йҮҸйҷҚдҪҺгҖӮеҰӮжһңеңЁTPSдёҠе№¶жІЎжңүиҝҮеӨ§зҡ„еҺӢеҠӣдҪҶжҳҜеҶ…еӯҳзҙ§дҝҸзҡ„жғ…еҶөдёӢе»әи®®иҝҳжҳҜ йҮҮз”ЁOpenj9зҡ„иҷҡжӢҹжңәгҖӮ
