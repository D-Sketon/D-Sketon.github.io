《大家的日语》文法整理

本笔记仅整理来自于《大家的日语》（みませんの日本語）1-22 课内的文法部分，可能会有疏漏的部分，敬请谅解

# 助词归纳

# は

表示其前面的名词是句子的主题（1.1）

  • 私は会社員です。

# も

用于将相同的事物作为前提加以陈述时（1.4）

  • 私は会社員です。ミラーさんも会社員です。

用于对疑问词所问范畴的一切全部加以否定时，动词使用否定（5.2）

  • どこも行きません。

# の

当名词 1 表示名词 2 的属性时，用来连接这两个名词（1.5）

  • ミラーさんは IMC の社員です。

当名词 1 用来说明名词 2 与什么事物有关系时，用来连接这两个名词（2.5）

  • これはコンピューターの本です。

当名词 1 表示名词 2 的所属（2.5）

  • これは私の本です。

答语中，代替问句中出现的名词，不能代替人（2.6）

  • このかばんはあなたのですか。・・・・・・いいえ、わたしのじゃありません。

表示方向和位置（10.4）

  • 机の上に写真があります。

# か

表示说话人不肯定或疑问的语气，构成疑问句（1.3）

  • ミラーさんは会社員ですか。

用于两个及两个以上的并列疑问句进行选择回答的选择疑问句，可翻译为 “或者，还是”（2.4）

  • これは「9」ですか、「7」ですか。

表示邀请人一起来做某事（6.6）

  • いっしょうに京都へ行きませんか。

说话人在得到自己所不知道的新信息之后表示认可、理解（6.8）

  • 日曜日京都へ行きました。・・・・・・京都ですか。いいですね。

# に

表示动作进行的时间（4.3）

  • 6 時半に起きます。

具体时间后需要加，如 6 時半、7 月 2 日
「~ 曜日」、「朝」、「昼」、「晩」、「夜」后可加可不加

表示授受动词接受 / 提供事物、信息的对象（非公司、学校等组织）（7.3/7.4）

  • イーさんに本を貸しました。
  • イーさんに本を借りました。

表示物和人存在的场所（10.2）

  • 私の部屋に机があります。
  • 机は私の部屋にあります。

表示频度（11.3）

  • 1 か月に 2 回映画を見ます。

表示目的（13.3）

  • 明日京都のお祭りに行きます。

表示主语因某一动作的结果而存在于某一场所（15.4）

  • ここに入ってはいけません。

表示动作的结果或名词存在的场所（15.5）

  • ここに車を止めてください。

表示变化的结果（19.4）

  • 元気になります。

# から

表示起点（时间、场所）（4.4）

  • 9 時から働きます。

表示授受动词接受 / 提供事物、信息的对象（公司、学校等组织必须用该助词）（7.4）

  • 銀行からお金を借りました。

表示原因（9.4）

  • 時間がありませんから、新聞を読みません。

表示动作相继进行（动词て形）（16.2）

  • この仕事が終わってから、食べます。

# まで

表示终点（时间、场所）（4.4）

  • 9 時から 5 時まで勉強します。

# と

表示名词并列，可翻译为 “和”（4.5）

  • 銀行の休みは土曜日と日曜日です。

表示与自己一起行动的人或动物（5.4）

  • 家族と日本へ来ました。

表示比较的对象（12.4）

  • 春と秋とどちらが好きですか。

表示思考、判断、发言的内容（21.1/21.2）

  • 明日雨が降ると思います。
  • 寝る前に、「おやすみなさい」と言います。

# ね

接在句尾，表示说话人希望得到听话人的同意或说话人对某一事物加以确认、叮嘱的语气（4.6）

  • 毎日 10 時まで勉強します。・・・・・・大変ですね。

# へ

表示移动的方向（5.1）

  • 日本へ来ました。

# て

用于连接两个及以上连续发生的动作（16.1）

  • 朝ジョギングをして、会社へ行きます。

用于连接两个以上句子，句末为い形容词，可翻译为 “并且”（16.1）

  • 昨日は天気が良くて、暑かったです。

# で

表示手段、方法（7.1）
－はしで食べます。

表示交通手段（5.3）

  • 電車で行きます。

徒步行走时，用歩いて，且不接で

  • 駅から歩いて帰りました。

表示动作发生的场所（6.5）

  • 駅で新聞を買います。

表示范围（12.5）

  • 日本料理で何が一番美味しいですか。

用于连接两个以上句子，句末为な形容词，可翻译为 “并且”（16.1）

  • 上海は賑やかで、きれいな町です。

用于连接两个以上句子，句末为名词（16.1）

  • 私は中国人で、学生です。

# よ

接在句尾，用于将对方不知道的事情或说话人自己的判断、意见等告诉对方（5.6）

  • この電車は京都へ行きますか。・・・・・・いいえ、行きません。次の「普通」ですよ。

# を

表示他动词的宾语（6.1）

  • ジュースを飲みます。

表示起点、出发点（16.4）

  • 7 時に家を出ます。

# が

表示转折（8.4）

  • 日本の食べ物は美味しいですが、高いです。

表示形容词和状态动词的宾语（9.1）

  • 私は日本語がわかります。
  • 私は日本語が好きです。
  • 私は友達が欲しいです。

表示作为存在主体的名词（10.1）

  • 桜があります。
  • 犬がいます。

表示五官所感的某种现象或客观地传达某件事情（14.6）

  • 雨が降っています。
  • ミラーさんがいませんね。

表示句子的主题具有的性质（16.3）

  • 大阪は食べ物が美味しいです。

# や

用于选择两个以上具有代表性的名词加以列举（10.5）

  • 箱の中に手紙や写真があります。

# など

表示除了列举出的以外还有其他名词，可翻译为 “等等”（10.5）

  • 箱の中に手紙や写真などがあります。

# だけ

接在数量词或名词后，表示只有这些，不会再有超过于此，可翻译为 “只有”（11.4）

  • 休みは日曜日だけです。

# より

表示比较（12.3）

  • この車はあの車より大きいます。

# ぐらい

接在数量词之后表示大概的数量，可翻译为 “大约”（11.2）

  • 15 分ぐらいかかります。

# でも

在劝诱或建议某件事时，提出某一示例但并不限定于此一项（21.6）

  • ちょっとビールでも飲みませんか。

# 表格归纳

# 指示代词一览表（3.5）

「こ」用于离说话人近
「そ」用于离听话人近
「あ」用于里说话人和听话人都比较远
「ど」用于疑问

こ系列そ系列あ系列ど系列
物体これそれあれどれ
物、人この名词その名词あの名词どの名词
场所ここそこあそこどこ
方向、场所（郑重）こちらそちらあちこどちら

# ます形时态（4.2）

ます形用于叙述现在的习惯、真理或将来会发生的事情、行动

非过去（现在、未来）过去
肯定起きます起きました
否定起きません起きませんでした

# 副词（9.3）

表示程度的副词表示数量的副词
和肯定形一起使用よくわかりますたくさんあります
だいたい少し
少し
和否定形一起使用あまりわかりませんあまりありません
全然全然

# 名词句、な形容词句时态（12.1）

非过去（现在、未来）过去
肯定名词雨です名词雨でした
な形容词静かな形容词静か
否定名词雨じゃ（では）ありません名词雨じゃ（では）ありませんでした
な形容词静かな形容词静か

# い形容词句时态（12.2）

非过去（现在、未来）过去
肯定暑いです暑かったです
否定暑くないです暑くなかったです
