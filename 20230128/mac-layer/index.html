ALOHA

  • з”ұеӨҸеЁҒеӨ·еӨ§еӯҰзҡ„Norm AbramsonејҖеҸ‘
  • е®Ңе…ЁеҲҶж•ЈеҚҸи®®

зәҜALOHA

  • еңЁзәҜALOHAдёӯпјҢз”ЁжҲ·еҸӘиҰҒжңүж•°жҚ®е°ұеҸ‘йҖҒеё§
  • з”ЁжҲ·еңЁеҸ‘з”ҹеҶІзӘҒеҗҺйҖүжӢ©йҡҸжңәзҡ„ж—¶й—ҙеҗҺйҮҚиҜ•
    • дҪҺиҙҹиҪҪдёӢзҡ„й«ҳж•ҲдҪҺе»¶иҝҹ
  • еҪ“е…¶д»–з”ЁжҲ·еңЁдёӨеҖҚдәҺеё§ж—¶й—ҙзҡ„и„Ҷејұж—¶ж®өеҶ…дј иҫ“ж—¶пјҢдјҡеҸ‘з”ҹеҶІзӘҒ
  • е°ҶеҸ‘йҖҒз«ҜдёҺж—¶й—ҙж§ҪеҗҢжӯҘеҸҜд»ҘеҮҸе°‘еҶІзӘҒ
    • её§ж—¶й—ҙпјҡеё§й•ҝеәҰйҷӨд»ҘжҜ”зү№зҺҮ
  • еҗһеҗҗйҮҸS=GГ—e(вҲ’2G)S = G \times e^ {(-2G)}S=GГ—e(вҲ’2G)
  • жңҖеӨ§еҗһеҗҗйҮҸеҮәзҺ°еңЁG=0.5пјҢS=1/2e=18.4%G = 0.5пјҢS = 1/2e = 18.4\%G=0.5пјҢS=1/2e=18.4%

еҲҶж§ҪALOHA

  • еҲҶж§ҪALOHAзҡ„ж•ҲзҺҮжҳҜзәҜALOHAзҡ„дёӨеҖҚ
  • дҪҺиҙҹиҪҪжөӘиҙ№ж—¶й—ҙж§ҪпјҢй«ҳиҙҹиҪҪеҜјиҮҙзў°ж’һ
  • еҗһеҗҗйҮҸS=GГ—e(вҲ’G)S = G \times e^ {(-G)}S=GГ—e(вҲ’G)
  • жңҖеӨ§еҗһеҗҗйҮҸеҮәзҺ°еңЁG=1пјҢS=1/e=36.8%G = 1пјҢS = 1/e = 36.8\%G=1пјҢS=1/e=36.8%

CSMA

  • CSMAйҖҡиҝҮж„ҹзҹҘдҝЎйҒ“ж”№иҝӣдәҶALOHA
    • еҰӮжһңз”ЁжҲ·ж„ҹи§үеҲ°е…¶д»–дәәпјҢеҲҷдёҚдјҡеҸ‘йҖҒ
    • вҖңе…Ҳж„ҹзҹҘеҗҺдј иҫ“вҖқжҲ–вҖңе…Ҳеҗ¬еҗҺиҜҙвҖқзҡ„еҺҹеҲҷ
  • 1-еқҡжҢҒCSMAжҢҒз»ӯдҫҰеҗ¬пјҢдёҖжңүз©әй—Із«ӢеҚіеҸ‘йҖҒ
  • йқһеқҡжҢҒCSMAеҰӮжһңдҫҰеҗ¬еӯҳеңЁиҪҪжіўпјҢеҲҷзӯүеҫ…йҡҸжңәж—¶й—ҙпјҢз„¶еҗҺйҮҚеӨҚдёҠиҝ°жӯҘйӘӨпјҢдјҡеёҰжқҘжӣҙеӨ§е»¶иҝҹпјҢдҪҶжңүжӣҙеҘҪзҡ„дҝЎйҒ“еҲ©з”ЁзҺҮ
  • p-еқҡжҢҒCSMAиҜ•з”ЁдёҺеҲҶж—¶й—ҙж§Ҫзҡ„дҝЎйҒ“гҖӮеҰӮжһңдҫҰеҗ¬ж—¶дҝЎйҒ“з©әй—ІпјҢеҲҷжҢүжҰӮзҺҮpеҸ‘йҖҒж•°жҚ®пјҢжҰӮзҺҮ1-pжҺЁиҝҹеҲ°дёӢдёҖдёӘж—¶й—ҙж§ҪпјҢеҰӮжһңеҸ‘з”ҹеҶІзӘҒпјҢеҲҷзӯүеҫ…йҡҸжңәж—¶й—ҙпјҢз„¶еҗҺйҮҚеӨҚдёҠиҝ°жӯҘйӘӨ

еёҰеҶІзӘҒжЈҖжөӢзҡ„CSMA

  • CSMA/CDзҡ„ж”№иҝӣжҳҜжЈҖжөӢ/дёӯжӯўеҶІзӘҒ
  • еҮҸе°‘з«һдәүж—¶й—ҙд»ҘжҸҗй«ҳжҖ§иғҪ
  • дёҖдёӘз«ҷеңЁеҸ‘йҖҒеё§еҗҺзӣ‘еҗ¬д»ӢиҙЁпјҢд»ҘжҹҘзңӢдј иҫ“жҳҜеҗҰжҲҗеҠҹ
    • еҰӮжһңжҳҜиҝҷж ·зҡ„иҜқпјҢиҝҷдёӘз«ҷе°ұз»“жқҹдәҶ
    • еҰӮжһңеҸ‘з”ҹзў°ж’һпјҢе°Ҷз«ӢеҚіз»ҲжӯўеҸ‘йҖҒпјҢйҡҸжңәзӯүеҫ…дёҖж®өж—¶й—ҙеҗҺеё§е°ҶеҶҚж¬ЎеҸ‘йҖҒ
  • дёҖдёӘз«ҷдј иҫ“дәҶ2П„еҗҺжІЎжңүзӣ‘еҗ¬еҲ°еҶІзӘҒпјҢеҲҷеҸҜд»ҘзЎ®дҝқиҮӘе·ұжҠ“дҪҸдәҶдҝЎйҒ“пјҲП„дёәдёӨдёӘзӣёи·қжңҖиҝңзҡ„з«ҷдј ж’ӯдҝЎеҸ·жүҖйңҖиҰҒзҡ„ж—¶й—ҙпјү

  • ејӮжҲ–иҝҷдәӣдҪҚпјӣз«ҷзӮ№еңЁеҸ‘йҖҒвҖң0вҖқдҪҶејӮжҲ–з»“жһңдёәвҖң1вҖқж—¶ж”ҫејғ

жңүйҷҗз«һдәүеҚҸи®®
