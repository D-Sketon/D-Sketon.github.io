1.е№¶еҸ‘жЎҶжһ¶Executor

  • JDK 5ејҖе§ӢжҸҗдҫӣExecutor FrameWork (java.util.concurrent.*)
    • еҲҶзҰ»д»»еҠЎзҡ„еҲӣе»әе’Ңжү§иЎҢиҖ…зҡ„еҲӣе»ә
    • зәҝзЁӢйҮҚеӨҚеҲ©з”Ё(newзәҝзЁӢд»Јд»·еҫҲеӨ§)

е…ұдә«зәҝзЁӢжұ 

  • йў„и®ҫеҘҪзҡ„еӨҡдёӘThreadпјҢеҸҜеј№жҖ§еўһеҠ 
  • еӨҡж¬Ўжү§иЎҢеҫҲеӨҡеҫҲе°Ҹзҡ„д»»еҠЎ
  • д»»еҠЎеҲӣе»әе’Ңжү§иЎҢиҝҮзЁӢи§ЈиҖҰ
  • зЁӢеәҸе‘ҳж— йңҖе…іеҝғзәҝзЁӢжұ жү§иЎҢд»»еҠЎиҝҮзЁӢ

дё»иҰҒзұ»

  • Executors.newCachedThreadPool/newFixedThreadPool еҲӣе»әзәҝзЁӢжұ 
  • ExecutorService зәҝзЁӢжұ жңҚеҠЎ
  • Callable е…·дҪ“зҡ„йҖ»иҫ‘еҜ№иұЎ(зәҝзЁӢзұ»)
    • пјҲRunnableзҡ„runж–№жі•жІЎжңүиҝ”еӣһеҖјпјҢиҖҢCallableзҡ„callж–№жі•еҸҜд»Ҙжңүиҝ”еӣһеҖјпјү
  • Future иҝ”еӣһз»“жһң

2.е№¶еҸ‘жЎҶжһ¶Fork-join

  • Java 7 жҸҗдҫӣеҸҰдёҖз§Қе№¶иЎҢжЎҶжһ¶пјҡеҲҶи§ЈгҖҒжІ»зҗҶгҖҒеҗҲе№¶(еҲҶжІ»зј–зЁӢ)
  • йҖӮеҗҲз”ЁдәҺж•ҙдҪ“д»»еҠЎйҮҸдёҚеҘҪзЎ®е®ҡзҡ„еңәеҗҲ(жңҖе°Ҹд»»еҠЎеҸҜзЎ®е®ҡ)

дё»иҰҒзұ»

  • ForkJoinPool д»»еҠЎжұ 
  • RecursiveAction
  • RecursiveTask

3.е№¶еҸ‘ж•°жҚ®з»“жһ„

  • еёёз”Ёзҡ„ж•°жҚ®з»“жһ„жҳҜзәҝзЁӢдёҚе®үе…Ёзҡ„

    • ArrayList, HashMap, HashSet йқһеҗҢжӯҘзҡ„
    • еӨҡдёӘзәҝзЁӢеҗҢж—¶иҜ»еҶҷпјҢеҸҜиғҪдјҡжҠӣеҮәејӮеёёжҲ–ж•°жҚ®й”ҷиҜҜ

  • дј з»ҹVectorпјҢHashtableзӯүеҗҢжӯҘйӣҶеҗҲжҖ§иғҪиҝҮе·®

  • е№¶еҸ‘ж•°жҚ®з»“жһ„пјҡж•°жҚ®ж·»еҠ е’ҢеҲ йҷӨ

    • йҳ»еЎһејҸйӣҶеҗҲпјҡеҪ“йӣҶеҗҲдёәз©әжҲ–иҖ…ж»Ўж—¶пјҢзӯүеҫ…
    • йқһйҳ»еЎһејҸйӣҶеҗҲпјҡеҪ“йӣҶеҗҲдёәз©әжҲ–иҖ…ж»Ўж—¶пјҢдёҚзӯүеҫ…пјҢиҝ”еӣһnullжҲ–ејӮеёё

  • List

    • Vector еҗҢжӯҘе®үе…ЁпјҢеҶҷеӨҡиҜ»е°‘
    • ArrayList дёҚе®үе…Ё
    • Collections.synchronizedList(List list) еҹәдәҺsynchronizedпјҢж•ҲзҺҮе·®
    • CopyOnWriteArrayList иҜ»еӨҡеҶҷе°‘пјҢеҹәдәҺеӨҚеҲ¶жңәеҲ¶пјҢйқһйҳ»еЎһ

  • Set

    • HashSet дёҚе®үе…Ё
    • Collections.synchronizedSet(Set set) еҹәдәҺsynchronizedпјҢж•ҲзҺҮе·®
    • CopyOnWriteArraySet (еҹәдәҺCopyOnWriteArrayListе®һзҺ°) иҜ»еӨҡеҶҷе°‘пјҢйқһйҳ»еЎһ

  • Map

    • Hashtable еҗҢжӯҘе®үе…ЁпјҢеҶҷеӨҡиҜ»е°‘
    • HashMap дёҚе®үе…Ё
    • Collections.synchronizedMap(Map map) еҹәдәҺsynchronizedпјҢж•ҲзҺҮе·®
    • ConcurrentHashMap иҜ»еӨҡеҶҷе°‘пјҢйқһйҳ»еЎһ

  • Queue & Deque (йҳҹеҲ—пјҢJDK 1.5 жҸҗеҮә)

    • ConcurrentLinkedQueue йқһйҳ»еЎһ
    • ArrayBlockingQueue/LinkedBlockingQueue йҳ»еЎһ

4.е№¶еҸ‘еҚҸдҪңжҺ§еҲ¶

Lock

  • Lockд№ҹеҸҜд»Ҙе®һзҺ°еҗҢжӯҘзҡ„ж•Ҳжһң
    • е®һзҺ°жӣҙеӨҚжқӮзҡ„дёҙз•ҢеҢәз»“жһ„
    • tryLockж–№жі•еҸҜд»Ҙйў„еҲӨй”ҒжҳҜеҗҰз©әй—І
    • е…Ғи®ёеҲҶзҰ»иҜ»еҶҷзҡ„ж“ҚдҪңпјҢеӨҡдёӘиҜ»пјҢдёҖдёӘеҶҷ
    • жҖ§иғҪжӣҙеҘҪ
  • ReentrantLockзұ»пјҢеҸҜйҮҚе…Ҙзҡ„дә’ж–Ҙй”Ғ
  • ReentrantReadWriteLockзұ»пјҢеҸҜйҮҚе…Ҙзҡ„иҜ»еҶҷй”Ғ
  • lockе’ҢunlockеҮҪж•°

Semaphore

  • дҝЎеҸ·йҮҸпјҡжң¬иҙЁдёҠжҳҜдёҖдёӘи®Ўж•°еҷЁ
  • и®Ўж•°еҷЁеӨ§дәҺ0пјҢеҸҜд»ҘдҪҝз”ЁпјҢзӯүдәҺ0дёҚиғҪдҪҝз”Ё
  • еҸҜд»Ҙи®ҫзҪ®еӨҡдёӘе№¶еҸ‘йҮҸпјҢдҫӢеҰӮйҷҗеҲ¶10дёӘи®ҝй—®
  • Semaphore
    • acquireиҺ·еҸ–
    • releaseйҮҠж”ҫ
  • жҜ”LockжӣҙиҝӣдёҖжӯҘпјҢеҸҜд»ҘжҺ§еҲ¶еӨҡдёӘеҗҢж—¶и®ҝй—®е…ій”®еҢә

Latch

  • зӯүеҫ…й”ҒпјҢжҳҜдёҖдёӘеҗҢжӯҘиҫ…еҠ©зұ»
  • з”ЁжқҘеҗҢжӯҘжү§иЎҢд»»еҠЎзҡ„дёҖдёӘжҲ–иҖ…еӨҡдёӘзәҝзЁӢ
  • дёҚжҳҜз”ЁжқҘдҝқжҠӨдёҙз•ҢеҢәжҲ–иҖ…е…ұдә«иө„жәҗ
  • CountDownLatch
    • countDown() и®Ўж•°еҮҸ1
    • await() зӯүеҫ…latchеҸҳжҲҗ0

Barrier

  • йӣҶеҗҲзӮ№пјҢд№ҹжҳҜдёҖдёӘеҗҢжӯҘиҫ…еҠ©зұ»
  • е…Ғи®ёеӨҡдёӘзәҝзЁӢеңЁжҹҗдёҖдёӘзӮ№дёҠиҝӣиЎҢеҗҢжӯҘ
  • CyclicBarrier
    • жһ„йҖ еҮҪж•°жҳҜйңҖиҰҒеҗҢжӯҘзҡ„зәҝзЁӢж•°йҮҸ
    • awaitзӯүеҫ…е…¶д»–зәҝзЁӢпјҢеҲ°иҫҫж•°йҮҸеҗҺпјҢе°ұж”ҫиЎҢ

Phaser

  • е…Ғи®ёжү§иЎҢе№¶еҸ‘еӨҡйҳ¶ж®өд»»еҠЎпјҢеҗҢжӯҘиҫ…еҠ©зұ»
  • еңЁжҜҸдёҖдёӘйҳ¶ж®өз»“жқҹзҡ„дҪҚзҪ®еҜ№зәҝзЁӢиҝӣиЎҢеҗҢжӯҘпјҢеҪ“жүҖжңүзҡ„зәҝзЁӢйғҪеҲ°иҫҫиҝҷжӯҘпјҢеҶҚиҝӣиЎҢдёӢдёҖжӯҘ
  • Phaser
    • arrive()
    • arriveAndAwaitAdvance()

Exchanger

  • е…Ғи®ёеңЁе№¶еҸ‘зәҝзЁӢдёӯдә’зӣёдәӨжҚўж¶ҲжҒҜ
  • е…Ғи®ёеңЁ2дёӘзәҝзЁӢдёӯе®ҡд№үеҗҢжӯҘзӮ№пјҢеҪ“дёӨдёӘзәҝзЁӢйғҪеҲ°иҫҫеҗҢжӯҘзӮ№пјҢе®ғд»¬дәӨжҚўж•°жҚ®з»“жһ„
  • Exchanger
    • exchange(), зәҝзЁӢеҸҢж–№дә’зӣёдәӨдә’ж•°жҚ®
    • дәӨжҚўж•°жҚ®жҳҜеҸҢеҗ‘зҡ„

Quartz

  • и§ЈеҶізЁӢеәҸдёӯTimerйӣ¶ж•Јз®ЎзҗҶзҡ„й—®йўҳ
  • еҠҹиғҪжӣҙеҠ ејәеӨ§
    • Timerжү§иЎҢе‘Ёжңҹд»»еҠЎпјҢеҰӮжһңдёӯй—ҙжҹҗдёҖж¬ЎжңүејӮеёёпјҢж•ҙдёӘд»»еҠЎз»Ҳжӯўжү§иЎҢ
